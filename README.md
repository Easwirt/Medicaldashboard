
  # Medical Dashboard Design

  This is a code bundle for Medical Dashboard Design. The original project is available at https://www.figma.com/design/bCbuO8dzZHOvgqYvqfbx5U/Medical-Dashboard-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run 'docker compose up -d --build backend' to build'

  Run `npm run dev` to start the development server.

  Project will be on http://localhost:5173/
